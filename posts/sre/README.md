# SRE

> Site Reliability Engineering practices — reliability, latency, incident response, and service health.

---

## Overview

Posts in this section cover SRE practices applied to Goalixa — from monitoring and alerting to incident response and post-mortem analysis. Learn how we balance reliability with feature velocity in a microservices environment.

---

## Topics Covered

- **Reliability**: Uptime, availability, SLIs/SLOs
- **Latency Analysis**: Performance optimization, user-perceived latency
- **Incident Response**: On-call procedures, escalation paths
- **Monitoring**: Metrics collection, alerting strategies
- **Post-Mortems**: Root cause analysis, prevention measures
