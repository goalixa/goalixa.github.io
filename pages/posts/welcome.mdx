# Welcome to the Goalixa Blog

Hi! I'm **Amirreza Rezaie**, and this is where I document the technical journey of building **Goalixa** — a personal productivity platform that helps people transform big goals into actionable plans, organized projects, and daily tasks.

This isn't just a product blog. It's an honest look at what it takes to build a modern cloud-native application from scratch — the architecture decisions, the mistakes, the incidents, and the lessons learned along the way.

---

## What This Blog Covers

I write about the practical side of building production software, organized into these topics:

### 🏗️ **Architecture & Design**
- Microservices architecture patterns
- Backend for Frontend (BFF) implementation
- API Gateway design
- System evolution from monolith to microservices

### 🔧 **DevOps & Infrastructure**
- Kubernetes orchestration
- GitOps with ArgoCD
- CI/CD pipelines (GitHub Actions → Harbor → ArgoCD)
- Container registry management
- Infrastructure as code

### 📊 **Observability & Reliability**
- Prometheus metrics collection
- Grafana dashboards
- Alertmanager configuration
- Production incident reports
- Root cause analysis

### 💻 **Engineering Practices**
- AI-assisted development with Claude
- Python microservices (Flask/FastAPI)
- Progressive Web App development
- Code quality and testing strategies

### 📖 **The Journey**
- Building under constraints (wartime internet restrictions in Iran)
- Real challenges and setbacks
- What worked, what didn't, and why
